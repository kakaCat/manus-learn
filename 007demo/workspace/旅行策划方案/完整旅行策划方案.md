## 五、注意事项

### 5.1 行前准备
1. **提前预订**：机票、酒店、热门景点门票建议提前1-2个月预订
2. **天气查询**：出发前一周查询目的地天气预报，调整衣物准备
3. **货币兑换**：提前兑换适量当地货币，了解汇率
4. **语言准备**：下载翻译APP，学习基本问候语
5. **健康检查**：如有特殊疾病，提前咨询医生并准备足够药品

### 5.2 安全事项
1. **证件安全**：重要证件拍照备份，原件与复印件分开存放
2. **财物安全**：不要携带大量现金，使用酒店保险箱
3. **人身安全**：避免夜间单独外出，了解当地紧急电话
4. **交通安全**：遵守当地交通规则，租车需购买全险
5. **食品安全**：注意饮食卫生，避免生冷食物

### 5.3 健康与医疗
1. **疫苗接种**：根据目的地要求提前接种疫苗
2. **医疗保险**：购买包含医疗运送的旅游保险
3. **高原反应**：如前往高原地区，提前做好适应准备
4. **防晒防暑**：热带地区注意防晒补水
5. **急救知识**：学习基本急救知识

### 5.4 文化与礼仪
1. **尊重习俗**：了解并尊重当地文化习俗和宗教信仰
2. **着装要求**：参观宗教场所时注意着装规范
3. **拍照礼仪**：拍照前征得同意，避免拍摄敏感场所
4. **小费文化**：了解当地小费习惯
5. **环保意识**：保护环境，不乱扔垃圾

### 5.5 通讯与联络
1. **国际漫游**：开通国际漫游或购买当地SIM卡
2. **紧急联系人**：告知家人行程安排和联系方式
3. **使馆信息**：记录中国驻当地使领馆联系方式
4. **网络连接**：了解酒店和公共场所WiFi情况
5. **重要号码**：保存当地报警、急救电话

## 六、应急处理

### 6.1 常见问题处理
1. **证件丢失**：立即报警并联系中国使领馆补办
2. **财物被盗**：报警并联系保险公司
3. **航班延误**：联系航空公司，了解补偿政策
4. **生病受伤**：立即就医，保留医疗单据
5. **语言障碍**：使用翻译APP或寻求酒店帮助
