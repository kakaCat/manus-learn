# 东京旅游指南

## 一、主要旅游景点

### 1. 历史文化景点
- **东京塔**：东京的标志性建筑，可俯瞰城市全景
- **皇居**：日本天皇的居所，外苑可免费参观
- **浅草寺**：东京最古老的寺庙，感受传统日本文化
- **明治神宫**：供奉明治天皇和昭宪皇后的神社
- **上野公园**：内有东京国立博物馆、上野动物园等

### 2. 现代都市景点
- **涩谷十字路口**：世界最繁忙的十字路口
- **新宿御苑**：东京最大的日式庭园和法式庭园
- **东京晴空塔**：世界第二高建筑，634米
- **台场**：人工岛，有彩虹桥、自由女神像复制品
- **六本木新城**：现代化综合设施，夜景绝佳

### 3. 购物娱乐区
- **银座**：高端购物区，奢侈品集中地
- **原宿**：年轻人时尚文化中心
- **秋叶原**：电器和动漫文化圣地
- **池袋**：购物和娱乐中心
- **表参道**：时尚品牌旗舰店集中地

### 4. 主题公园
- **东京迪士尼乐园**：亚洲第一座迪士尼乐园
- **东京迪士尼海洋**：全球唯一的海洋主题迪士尼
- **三鹰之森吉卜力美术馆**：宫崎骏动画主题博物馆

## 二、住宿推荐

### 1. 高端酒店（4-5星）
- **东京安缦酒店**：顶级奢华，皇居旁
- **东京柏悦酒店**：《迷失东京》取景地
- **东京半岛酒店**：银座核心位置
- **东京文华东方酒店**：日本桥地区
- **东京丽思卡尔顿酒店**：六本木地区

### 2. 中档酒店（3-4星）
- **东京新宿华盛顿酒店**：交通便利
- **东京池袋太阳城王子酒店**：购物方便
- **东京银座蒙特利酒店**：地理位置优越
- **东京涩谷东急酒店**：年轻人聚集区
- **东京上野三井花园酒店**：靠近上野公园

### 3. 经济型住宿
- **胶囊旅馆**：体验日本特色住宿
- **商务酒店**：如东横INN、APA酒店
- **青年旅舍**：适合背包客
- **民宿**：Airbnb等平台预订
- **网吧**：24小时营业，可过夜

## 三、当地交通信息

### 1. 主要交通方式
- **JR山手线**：环状线路，连接主要车站
- **东京地铁**：13条线路，覆盖全市
- **都营地铁**：4条线路
- **私铁**：如东急、京王、小田急等
- **巴士**：补充地铁网络

### 2. 交通卡
- **Suica卡**：JR东日本发行的IC卡
- **Pasmo卡**：私铁和地铁公司发行的IC卡
- **东京地铁通票**：1日/2日/3日券
- **JR东京广域周游券**：适合周边游

### 3. 主要车站
- **东京站**：新干线主要车站
- **新宿站**：世界最繁忙车站
- **涩谷站**：年轻人聚集地
- **池袋站**：购物娱乐中心
- **上野站**：前往东北地区门户

### 4. 机场交通
- **成田机场**：N'EX成田特快、京成Skyliner
- **羽田机场**：东京单轨电车、京急电铁

## 四、适合中国游客的旅行注意事项

### 1. 签证信息
- **旅游签证**：需通过指定旅行社办理
- **所需材料**：护照、照片、申请表、在职证明、资产证明等
- **签证类型**：单次、三年多次、五年多次
- **办理时间**：通常5-10个工作日
- **费用**：根据签证类型不同

### 2. 货币与支付
- **货币**：日元（JPY）
- **汇率**：1人民币≈20日元（参考）
- **支付方式**：
  - 现金：仍广泛使用，特别是小商店
  - 信用卡：大型商场、酒店、餐厅
  - 移动支付：支付宝、微信支付在部分商家可用
  - IC卡：Suica/Pasmo可用于购物

### 3. 语言沟通
- **官方语言**：日语
- **英语普及度**：旅游区基本可用
- **中文服务**：
  - 大型商场、免税店有中文店员
  - 旅游景点有中文导览
  - 部分餐厅有中文菜单
- **实用APP**：
  - Google翻译（可拍照翻译）
  - 日本地铁
  - 乘换案内

### 4. 文化礼仪
- **鞠躬**：常见问候方式
- **脱鞋**：进入传统日式房间需脱鞋
- **温泉礼仪**：先淋浴再泡汤，禁止穿泳衣
- **用餐礼仪**：
  - 不要将筷子插在饭中
  - 吃面时可以发出声音
  - 不要自己给自己倒酒
- **排队文化**：严格遵守排队秩序

### 5. 安全与健康
- **治安**：东京治安良好，但仍需注意财物安全
- **医疗**：
  - 购买旅游保险
  - 大型医院有国际部
  - 药妆店可购买常用药品
- **紧急电话**：
  - 警察：110
  - 救护车/消防：119
  - 日本观光厅游客热线：03-3201-3331（中文）

### 6. 网络与通讯
- **WiFi**：
  - 机场、车站、便利店有免费WiFi
  - 可租用移动WiFi设备
  - 购买日本SIM卡
- **电压**：100V，两孔扁插头
- **时差**：比北京时间快1小时

### 7. 购物退税
- **免税条件**：同一店铺单日购物满5000日元
- **免税商品**：一般物品、消耗品
- **退税方式**：店内直接免税或机场退税
- **免税手续**：需出示护照

### 8. 饮食注意事项
- **自来水**：可直接饮用
- **饮食习惯**：
  - 早餐相对简单
  - 午餐有定食套餐
  - 晚餐较丰富
- **特色美食**：
  - 寿司、刺身
  - 拉面、乌冬面
  - 天妇罗、炸猪排
  - 和牛、烤肉

### 9. 最佳旅行时间
- **春季（3-5月）**：樱花季，气候宜人
- **夏季（6-8月）**：炎热潮湿，有花火大会
- **秋季（9-11月）**：红叶季，天气凉爽
- **冬季（12-2月）**：相对寒冷，有圣诞灯饰

### 10. 实用建议
1. **提前规划**：热门景点和餐厅需提前预订
2. **交通高峰**：避开早晚高峰（7:30-9:30，17:30-19:30）
3. **垃圾分类**：注意垃圾分类规则
4. **小费**：日本没有小费文化
5. **吸烟**：指定吸烟区吸烟，公共场所禁烟

## 五、推荐行程安排

### 3日经典游
- **第1天**：浅草寺 → 东京晴空塔 → 秋叶原
- **第2天**：皇居 → 银座 → 东京塔 → 六本木
- **第3天**：涩谷 → 原宿 → 新宿

### 5日深度游
- **第1天**：上野公园 → 浅草寺 → 东京晴空塔
- **第2天**：迪士尼乐园或迪士尼海洋
- **第3天**：皇居 → 银座 → 东京塔
- **第4天**：涩谷 → 原宿 → 表参道 → 新宿
- **第5天**：台场 → 秋叶原

### 7日周边游
- 前5天参考深度游
- **第6天**：镰仓一日游（灌篮高手取景地）
- **第7天**：箱根一日游（温泉、富士山）

- 餐饮：400-600元/天
- 交通：150-200元/天
- 门票：800-1200元
- **总计**：10000-15000元

### 豪华型（5-7天）
- 机票：5000-8000元
- 住宿：2000-4000元/晚
- 餐饮：800-1500元/天
