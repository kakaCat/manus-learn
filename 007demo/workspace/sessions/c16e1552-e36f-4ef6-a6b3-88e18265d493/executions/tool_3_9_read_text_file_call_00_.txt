东京旅游景点和活动快速查询指南

【必游景点TOP10】
1. 东京塔 - 标志性建筑，观景台
2. 浅草寺 - 最古老寺庙，雷门
3. 皇居 - 天皇居所，外苑免费
4. 东京晴空塔 - 634米，世界第二高
5. 涩谷十字路口 - 世界最繁忙
6. 上野公园 - 博物馆、动物园
7. 迪士尼乐园/海洋 - 主题公园
8. 秋叶原 - 电器动漫圣地
9. 银座 - 高端购物区
10. 台场 - 人工岛，彩虹桥

【美食推荐】
★ 寿司/刺身：新鲜海鲜
★ 拉面：一兰、一风堂等
★ 天妇罗：炸虾蔬菜
★ 和牛：神户牛、松阪牛
★ 抹茶甜品：冰淇淋、蛋糕
★ 居酒屋：体验日本酒文化

【购物地点】
◆ 药妆店：松本清、大国药妆、唐吉诃德
◆ 电器：秋叶原、Bic Camera、Yodobashi
◆ 百货：三越、伊势丹、高岛屋
◆ 时尚：银座、原宿、表参道
◆ 平价：优衣库、无印良品、GU

【活动体验】
☆ 文化：茶道、和服、寿司制作
☆ 季节：樱花(3-4月)、花火(7-8月)、红叶(10-11月)
☆ 夜景：东京塔、六本木
☆ 温泉：周边一日游

【实用信息】
✓ 交通：地铁+JR山手线，Suica/Pasmo卡
✓ 支付：现金+支付宝/微信+银联卡
✓ 退税：单店单日满5000日元
✓ 语言：旅游区有中文服务，下载翻译APP
✓ 紧急：警察110，救护119，使馆03-3403-3388

【推荐行程】
● 3日经典：浅草寺→晴空塔→秋叶原 / 皇居→银座→东京塔 / 涩谷→原宿→新宿
● 5日深度：+迪士尼+台场
● 购物专线：药妆店→电器街→百货商场

【预算参考】
经济型：6000-9000元(5-7天)
舒适型：10000-15000元(5-7天)
豪华型：20000-35000元(5-7天)

【中国游客贴士】
1. 签证：通过旅行社办理
2. 现金：每人每天5000-10000日元
3. 网络：租WiFi或买SIM卡
4. 保险：必须购买旅游保险
5. 礼仪：遵守排队、安静、整洁

最佳时间：春季(3-5月)樱花，秋季(9-11月)红叶
避开高峰：早晚交通(7:30-9:30, 17:30-19:30)
